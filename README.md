# 🎮 Video Game Sales Analysis Dashboard

## 📌 Project Overview
This project analyzes global video game sales data across multiple gaming platforms, genres, publishers, and regions using SQL and Power BI.

## 🛠 Tools Used
- MySQL
- Power BI
- SQL
- Advanced Excel

## 📊 Key Analysis
- Regional sales trends
- Genre analysis
- Platform performance
- Publisher analysis
- Year-wise sales trends

## 📈 Dashboard Features
- Interactive KPIs
- Filters & slicers
- Regional comparison
- Sales insights
